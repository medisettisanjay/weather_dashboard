Live City Weather Dashboard

The Live City Weather Dashboard is an interactive data visualization project designed to monitor and analyze real-time weather conditions across multiple cities. The dashboard integrates live weather data using the OpenWeather API and presents key environmental metrics through dynamic charts, KPI cards, maps, and tables.

Dashboard Features
Real-time weather monitoring for multiple cities
KPI cards displaying:
Average Temperature
Average Humidity
Average Pressure
Average Wind Speed
Interactive city selection filter
Temperature comparison bar chart between cities
Geographic map visualization showing city-wise temperature distribution
Detailed weather data table including:
Temperature
Wind Speed
Pressure
Humidity
Weather Description
Automatic data refresh with latest timestamp updates
Technologies Used
Power BI
OpenWeather API
Data Visualization
Data Modeling
DAX Functions
Key Insights
Compare weather conditions across different cities in real time
Identify temperature variations and climate patterns
Analyze humidity, pressure, and wind speed trends
Visualize geographic weather distribution using maps
